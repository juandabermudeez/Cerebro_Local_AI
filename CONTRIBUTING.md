# ✨ Contribuyendo a Cerebro Local AI

¡Gracias por tu interés en mejorar este proyecto! Este es un sistema personal, pero las ideas y mejoras de la comunidad son bienvenidas.

## 🚀 ¿Cómo puedo contribuir?

### 1. Reportar Errores
Si encuentras un bug, por favor abre un **Issue** describiendo:
- El comportamiento esperado.
- El comportamiento actual.
- Pasos para reproducir el error.

### 2. Sugerir Funcionalidades
¿Tienes una idea para una nueva visualización en el dashboard o un comando para el bot? Abre un **Issue** con la etiqueta `enhancement`.

### 3. Enviar Pull Requests
Si ya tienes una mejora lista:
1. Haz un **Fork** del repositorio.
2. Crea una rama para tu mejora (`git checkout -b feature/mejora-increible`).
3. Realiza tus cambios y haz commit (`git commit -m 'Añade funcionalidad X'`).
4. Sube tus cambios (`git push origin feature/mejora-increible`).
5. Abre un **Pull Request**.

## 🛠️ Guía de Desarrollo

### Estilo de Código
- Sigue las convenciones de **PEP 8**.
- Usa comentarios claros para explicar la lógica de la base de datos o el manejo de mensajes de Telegram.

### Pruebas
Asegúrate de que el bot y el dashboard funcionen correctamente antes de enviar tu PR. Verifica que no hayas incluido archivos sensibles (como tu `.env` o la base de datos real).
